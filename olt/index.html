弱光检测相关命令

查看槽位弱光检测状态
show pon onu optical-power-detect detail x
// 查看x槽位有无弱光检测
设置弱光告警阈值
onu optical-power-detect gpon olt-1/x/x online min-rx-power -28
// 设置弱光28需要在PON模式下
关闭弱光检测
onu optical-power-detect gpon olt-1/x/x disable
// 关闭弱光检测
💡 提示： 配置弱光告警时需要在PON模式下进行，x/x表示具体的端口号，请根据实际情况替换。
