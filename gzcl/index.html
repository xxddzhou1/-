故障处理

长发光ONU 的检测关闭
rogue-onu-detect olt gpon_olt-1/1/1 disable
// 在全局模式下进入pon模式输入命令，关闭1/1/1口发光检测
同台OLT下面固话语音单通
security port-protect dis
security user-communication control enable
security user-communication svlan 语音VLAN
// 局模式下通过show service-port interface gpon_onu-1/12/4:1查看MDU设备的语音vlan
💡 提示： 注：只有同一台OLT的语音单通有用，不是同一台olt做这个没用
